# SR Ludo Live Prototype

A front-end prototype implementing a dark-themed real-time Ludo battle flow with:

- Smart dual-wallet entry deduction
- Rules modal, room-code exchange, and result flow
- Referral analytics with 2% lifetime commission
- KYC form locking and admin-contact flow
- Sidebar history modals (match, deposit, withdraw)

## Run

```bash
python3 -m http.server 4173
```

Open <http://localhost:4173>.
